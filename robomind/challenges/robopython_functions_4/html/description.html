The Letter C

To complete our Letter Library we add a function function_c that draws the letter C. We also add a function goto_line that moves Robo to the beginning of a line. Test this with the following code: 

    robo.goto_line(1)
    robo.function_a()
    robo.function_b()
    robo.function_c()
    robo.goto_line(0)
    robo.function_c()
    robo.function_b()
    robo.function_a()
