Break the Loop

The break statement makes it possible to prematurely terminate a loop (for or while). When Python encounters a break, it stops processing the current loop. Processing continues with the first line of code after the loop.

As an example, the following code walks forward while there is a wall to the right (robo.rightIsObjstacle() is True) and there is no obstacle in front (robo.frontIsObstacle() is False). Stop immediately when there is something in front of Robo.

while robo.rightIsObstacle(): if not robo.frontIsObstacle(): robo.forward() else: break
