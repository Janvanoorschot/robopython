Robo's Commmands

Just like in the Robomind language, in the Python language you can instruct Robo to do something by giving Robo instructions. In the Robomind language you would do that by just include the instruction in your script like this:

forward

In the Python language you give Robo the same instruction by using the following line in your script:

robo.forward()

Press to Run button to see some of the instructions in action. Open the Help section below and have a look at all instructions there are to instruct Robo.
