Functions

A function contains a block of reusable code that performs a certain action. Using a function means calling the function with a number of parameters. An example function is: 

def my_function(arg1, arg2):
    return arg1 + arg2

This very simple function called my_function can be used (called) like this: 

    result5 = my_function(2,3)
