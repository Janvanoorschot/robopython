Teaching Robo to talk Python

Typically Robo talks the Robomind language. The Robomind language is simple and allows you to just type 'forward' and Robo will take one step forward. Using a simple language allows you to concentrate on learning Computational Thinking.

You can also use the Python language to program Robo. is a powerful language used for just about everything. Just start your program with the line '#!python' and Robo will know you are talking in the Python language (eat your heart out Harry).

Just press the Run button and Robo will execute your first Python program. Have a look at the Python code and try to recognise some familiar components.
