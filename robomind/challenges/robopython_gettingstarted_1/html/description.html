Compare Robo with Python

Robo talks Python language. Start the program with the lines '#!python', followed by a line 'global robo':

The following table gives the standard robo commands and their Python equivalent:

Basic Robo Instructions and their Python Equivalents

Robo Python Function
forward robo.forward(steps)Go one step forward. Use forward(number) to do more steps at once.
backward robo.backward(steps)Go one step backward. Use backward(number) to do more steps at once.
left robo.left(count)Make a quarter turn to the left. Use left(number) to do more steps at once.
right robo.right(count)Make a quarter turn to the right. Use right(number) to do more steps at once.
pickUp robo.pickUp()Get the beacon in front of the robot.
putDown robo.putDown()Put a beacon in front of the robot.
eatUp robo.eatUp()Pick up and destroy the beacon in front.
paintWhite robo.paintWhite()Put the brush with white paint to the ground.
paintBlack robo.paintBlack()Put the brush with black paint to the ground.
stopPainting robo.stopPainting()Stop painting, hide the brush.
leftIsClear robo.leftIsClear()The location to the left of Robo is empty.
leftIsObstacle robo.leftIsObstacle()The location to the left of Robo is not empty.
leftIsBeacon robo.leftIsBeacon()The location to the left of Robo contains a beacon.
leftIsWhite robo.leftIsWhite()The location to the left of Robo is painted white.
leftIsBlack robo.leftIsBlack()The location to the left of Robo is painted black.
frontIsClear robo.frontIsClear()The location in front of Robo is empty.
frontIsObstacle robo.frontIsObstacle()The location in front of Robo is not empty.
frontIsBeacon robo.frontIsBeacon()The location in front of Robo contains a beacon.
frontIsWhite robo.frontIsWhite()The location in front of Robo is painted white.
frontIsBlack robo.frontIsBlack()The location in front of Robo is painted black.
rightIsClear robo.rightIsClear()The location to the right of Robo is empty.
rightIsObstacle robo.rightIsObstacle()The location to the right of Robo is not empty.
rightIsBeacon robo.rightIsBeacon()The location to the right of Robo contains a beacon.
rightIsWhite robo.rightIsWhite()The location to the right of Robo is painted white.
rightIsBlack robo.rightIsBlack()The location to the right of Robo is painted black.
flipCoin robo.flipCoin()Let Robo flop a Coin. Return True if heads, False otherwise.

Basic Python Constructs

Loops

"for" loop

robo.paintWhite() for i in range (4): robo.forward(3) robo.right(1)

"while" loop

while True: if robo.frontIsObstacle(): quit() else: robo.forward(1)

"break" en "continue" statements

count = 0 while True: robo.frontIsObstacle(count): count += 1 if count >= 5: break
