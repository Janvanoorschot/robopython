Where are the {}?

In the Robomind language, we use curly braces ('{}') to seperate the instructions of, for example, the instructions in a 'repeat' block.

repeat(3) { forward }

In the Python language no curly braces ('{}') are used. Instead the line is ended with a colon (':'), and the instructions in the block are indended with spaces, like this:

while not robo.frontIsBeacon(): robo.forward()
