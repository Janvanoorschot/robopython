Variables

Variables are used in Python to store data for later use. For example: 

  variable_1 = 10
  variable_1 = variable_1 - 1
  if variable_1 == 9:
    robo.message("yes!!!")

Lots of different types of data can be stored in a variables such as:

  • A number such as 1, 2 or 3
  • A string such as "hello world"
  • A boolean such as True or False
  • A reference to a function such as paint_line
  • A reference to an object such as robo
  • A list of things such as [1, "hello world", robo]
