While loop

A while loop is very similar to an if statement. Both statements only execute the corresponding code if an equation is True. The difference is that with an if statement this code is only executed once. With a while loop, the code runs as long as the equation remains True.

Sometimes you want to escape the while loop before the current code block is done. The break instruction does just that. The continue statement will stop the execution of current code block and restart the code block. Both instructions also work in a for loop.
