If statement

Like the Robomind language, Python provides an if statement to conditionaly execute statements. The if in Python looks like this: 

if x < 0:
   x = 0
   print('Negative changed to zero')
elif x == 0:
   print('Zero')
elif x == 1:
   print('Single')
else:
   print('More')

No curly braces and an extra construct elif that is short for the Robomind language else if, but for the rest ... looks familiar!!.

Create four if/else statements that check if there is an obstacle in front of Robo. Move backward if there is, move forward if there is none.

The conditions of the if statement are so-called logical expressions. Such an expression will result in the value True or False in Python (note the capital 'T' and 'F') which is then used to decide to step to the appropriate part of the code to resume execution. Basic instructions may also be composed with the boolean operators not, and, or.
