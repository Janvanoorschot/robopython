Dictionaries

A dictionary is a collection of data items where each item data value is stored under a certain unique key. To find an data value, you must know the key of that value. Create a dictionary as follows: 

    dict1 = {"key1": 1, "key2": 2, "key3": 101 }
    dict2 = {1: "value1", 2: "value2", 3: "value3" }

You can access items in a dictionaly by using the key as index as follows: 

    my_int = dict1["key2"]
    if my_int == 2:
        robo.message("yes!!!!")
