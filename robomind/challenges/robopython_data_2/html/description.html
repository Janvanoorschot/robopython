Lists

A list is an ordered set of data variables. The following shows the creation of a couple of lists. 

    list1 = [1,2,3]
    list2 = ["help", "me", "please"]
    list3 = [robo, 1, "help]

you can access items from a list by using a so-called 'index' as follows: 

    my_int = list[1]
    if my_int == 2:
       robo.message("yes!!!")
