Getting started with RoboMind

Robo talks Python language. Start the program with the lines '#!python', followed by a line 'global robo':

The following table gives the standard robo commands and their Python equivalent:

Robo Python
forward robo.forward(steps)
backward robo.backward(steps)
left robo.left(count)
right robo.right(count)
pickUp robo.pickUp()
putDown robo.putDown()
eatUp robo.eatUp()
paintWhite robo.paintWhite()
paintBlack robo.paintBlack()
stopPainting robo.stopPainting()
leftIsClear robo.leftIsClear()
leftIsObstacle robo.leftIsObstacle()
leftIsBeacon robo.leftIsBeacon()
leftIsWhite robo.leftIsWhite()
leftIsBlack robo.leftIsBlack()
frontIsClear robo.frontIsClear()
frontIsObstacle robo.frontIsObstacle()
frontIsBeacon robo.frontIsBeacon()
frontIsWhite robo.frontIsWhite()
frontIsBlack robo.frontIsBlack()
rightIsClear robo.rightIsClear()
rightIsObstacle robo.rightIsObstacle()
rightIsBeacon robo.rightIsBeacon()
rightIsWhite robo.rightIsWhite()
rightIsBlack robo.rightIsBlack()
flipCoin robo.flipCoin()
