Move the beacon

Let's start with a simple Python program that gives Robo a sequence of simple commands.

Pick up the beacon and put it down in the lower left corner of the map. Use the following instructions: (scroll down to the 'Basic Instructions' section of Help for a complete list): 

    robo.right()
    robo.forward()
    robo.pickUp()
    robo.putDown()

Like with the Robomind language, in Python you can pass arguments to each instruction. Most of the instructions (also called 'methods' or 'functions') that Robo understand accept a single number as argument. For example, the following instruction tells Robo to move three paces forwards: 

    robo.forward(3)
