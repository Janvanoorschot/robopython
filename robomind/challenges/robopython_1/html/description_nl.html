Aan de slag met RoboMind

Robo kan ook Python spreken. De eerste regel van het programma moet zijn '#!python', de tweede 'global robo'.

De volgende tabel geeft de standaard robo commando's en de Python equivalenten:

Robo Python
vooruit robo.forward(steps)
achteruit robo.backward(steps)
links robo.left(count)
rechts robo.right(count)
pakOp robo.pickUp()
zetNeer robo.putDown()
eetOp robo.eatUp()
verfWit robo.paintWhite()
verfZwart robo.paintBlack()
stopVerven robo.stopPainting()
linksIsVrij robo.leftIsClear()
linksIsObstakel robo.leftIsObstacle()
linksIsBaken robo.leftIsBeacon()
linksIsWit robo.leftIsWhite()
linksIsZwart robo.leftIsBlack()
voorIsVrij robo.frontIsClear()
voorIsObstakel robo.frontIsObstacle()
voorIsBaken robo.frontIsBeacon()
voorIsWit robo.frontIsWhite()
voorIsZwart robo.frontIsBlack()
rechtsIsVrij robo.rightIsClear()
rechtsIsObstakel robo.rightIsObstacle()
rechtsIsBaken robo.rightIsBeacon()
rechtsIsWit robo.rightIsWhite()
rechtsIsZwart robo.rightIsBlack()
kopOfMunt robo.flipCoin()
