Aan de slag met RoboMind

Robo kan bewegen, verven en dingen oppakken. Laten we eens kijken hoe we Robo dit kunnen laten doen ...

Onderin het scriptvenster zie je een soort afstandsbediening waarmee je de robot instructies kan geven. Je kunt die natuurlijk ook direct zelf intypen.

Rechts onder de robot zie je een baken. Laat de robot het baken oppakken door hem de juiste instructies te geven.

Gebruik het scriptvenster om de robot instructies te geven. Druk op om het resultaat te zien. Druk op om stap voor stap door je script heen te gaan.
