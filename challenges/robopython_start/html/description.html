Getting started with RoboMind

Robo can move, paint and pick up things. Let's have a look how we can make Robo to do these things....

At the bottom of the script editor you see a kind of remote control with which you can give instructions to the robot. Of course you can also directly write commands in the script editor.

To the lower right of the robot there is a beacon. Let the robot pick it up by giving him the correct commands.

Use the script editor to give the robot commands. Push to see the result. You can also push to go through your script step-by-step.
